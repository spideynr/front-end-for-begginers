Web Development
Why we are here

Front-end Web development mainly consists of 3 parts

  • HTML
  • CSS
  • Javascript

HyperText Markup Language (HTML)

  • Barebones structure of a web page
  • Simple language made of elements

Cascading Style Sheets (CSS)

  • Handles the look and feel of html elements
  • Helps separate styling logic from html

Javascript

  • Handles the interaction logic between a user and the browser
  • Allows for implementation of complex things on web pages ( Dynamically controlling content etc)

    Styling html elements

    There is also an attribute and an element called style in HTML markup

    Color in HTML

    Color is represented in multitude of ways in HTML. RGB, RGBA, HSL, HSLA, HEX and some default colors already specified in each browser

    Named colors

    
            color: red;
            color: orange;
            color: tan;
            color: rebeccapurple;
            ...
            etc

    Hex colors

    
            color: #090;
            color: #009900;
            color: #090a;
            color: #009900aa;

    RGB colors

    
            color: rgb(34, 12, 64, 0.6);
            color: rgba(34, 12, 64, 0.6);
            color: rgb(34 12 64 / 0.6);
            color: rgba(34 12 64 / 0.3);
            color: rgb(34.0 12 64 / 60%);
            color: rgba(34.6 12 64 / 30%);

    HSL colors

    
            color: hsl(30, 100%, 50%, 0.6);
            color: hsla(30, 100%, 50%, 0.6);
            color: hsl(30 100% 50% / 0.6);
            color: hsla(30 100% 50% / 0.6);
            color: hsl(30.0 100% 50% / 60%);
            color: hsla(30.2 100% 50% / 60%);

    When you need to style an element you need to be able to select it first, and then instruct the browser to apply a given styling property
